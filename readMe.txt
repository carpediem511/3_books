## Проект "Книжная полка"

### ОСНОВНАЯ ИДЕЯ

Основная идея заключается в создании веб-приложения для управления списком книг. 
Пользователи могут просматривать, добавлять, редактировать и удалять книги. 
Это приложение обеспечивает простой и удобный способ управления библиотекой книг.

### ОСНОВНЫЕ КОМПОНЕНТЫ ПРОЕКТА

1. **Веб-страница со списком книг:** отображается список книг, который динамически заполняется с использованием JavaScript. 
Каждая книга содержит информацию о названии, авторе, годе издания и изображении.

2. **Форма добавления книги:** Пользователи могут добавлять новые книги, указывая название, автора, год издания и ссылку на изображение. 
После ввода данных и нажатия кнопки "Сохранить", новая книга добавляется в список.

3. **Функциональность редактирования книг:** При нажатии на кнопку "Обновить" рядом с книгой, открывается модальное окно, 
позволяющее изменить данные о книге. Пользователи могут изменить название, автора, год издания и изображение, 
а затем сохранить обновленные данные.

4. **Функциональность удаления книги:** Рядом с каждой книгой имеется кнопка "Удалить". При её нажатии, 
соответствующая книга удаляется из списка.

### ИСПОЛЬЗУЕМЫЕ ТЕХНОЛОГИИ

- **HTML и CSS:** Для разметки веб-страницы и создания стилей, обеспечивающих привлекательный дизайн.

- **JavaScript:** Для динамического добавления, редактирования и удаления книг на странице, 
а также для взаимодействия с пользовательскими данными.

- **Модальные окна:** Для реализации функциональности редактирования книг.

- **Local Storage:** Для сохранения изменений на сайте, чтобы пользователи могли продолжить работу с библиотекой 
после перезагрузки страницы.

### ОСОБЕННОСТИ ПРОЕКТА

- Возможность добавления, редактирования и удаления книг на странице, что делает приложение удобным для управления библиотекой.

- Интуитивно понятный интерфейс, который позволяет пользователям легко взаимодействовать с приложением.

### ИНСТРУКЦИИ ПО УСТАНОВКЕ

1. Клонируйте репозиторий с проектом на свой компьютер.

2. Откройте файл `index.html` в вашем веб-браузере.

3. Теперь вы можете взаимодействовать с приложением, добавлять, редактировать и удалять книги, а также оценить его функциональность.
